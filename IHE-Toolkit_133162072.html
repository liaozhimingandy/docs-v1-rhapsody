IHE Toolkit

The IHE Toolkit is a licensed add-on module for the Rhapsody Integration Engine, which is designed to simplify the process of implementing a system that complies with IHE Integration Profiles.

Refer to IHE Toolkit for product documentation on the IHE Toolkit.
