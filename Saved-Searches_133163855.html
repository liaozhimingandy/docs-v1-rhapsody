Saved Searches

Menu Path : Find Messages>Saved Searches

Read Access Rights:

  • Log in to Management Console
  • View locker
  • Search message queues
  • Search Error and Hold Queues

Write Access Rights:

  • Save searches
  • Send searches

Rhapsody enables you to save search queries so you can easily re-run a search. The Saved Searches page enables you to view all your saved searches and searches that have been shared with you by other users.

As saved searches are unique for each user logged into the Management Console, you can share saved searches with other users by sending the saved searches to them.

To save search queries, you must have 'Save searches' access right. To send saved queries, you must have 'Send searches' access right.


Column

Description
Warning
The warning icon is displayed when you do not have the all the required permissions to run the search. This could happen if another user sent you the search, or if your permissions have changed. The search can still be executed, but the results would be limited based on the permissions you do have.

Name

Displays the name of the search query. Click the Rename link to rename the search query.

Search

Contains a View link to load the search.

Saved

Displays the time since the search query was last saved. On mouse hover, a tooltip is displayed showing the actual date and time the query was last saved.

To send a search:

  1. Select the searches you wish to send. The Send Search and Delete buttons in the toolbar will now be enabled.

    The user must have permission to send a search.

  2. Select the Send Search button to display the Send Search dialog:

  3. Select the appropriate user or users.

    If the user you are sending the search to does not have the required access rights to run the search, a warning confirmation dialog is displayed.

  4. Select the Send button to complete the send operation.
