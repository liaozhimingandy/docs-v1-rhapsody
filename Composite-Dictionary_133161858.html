Composite Dictionary

The Composite Dictionary tab in the X12 Project Editor enables you to define composite data elements. A composite data element typically consists of two or more simple data elements.

The Composite Dictionary tab displays the following information:

Column Description
ID The composite data element number.
Description The user-defined description of the composite data element. By default, this field is set to the composite data element name.
Data Elements The data elements comprising the composite data element.

You can perform the following actions through the Composite Dictionary tab:

Action
Description
Add Create a new composite data element
Edit Edit the composite data element.
Delete Delete the selected composite data element.
Filter Filter the displayed composite data elements using a text-based filter.

Adding a Composite Data Element

To add a composite data element:

  1. Select the Add button.
  2. Select the OK button to save your changes, or the Cancel button to discard them.

Viewing the Structure of a Composite Data Element

To view the structure of a composite data element, select the Browse icon adjacent to it to display the Composite Data Element dialog:

The Composite Data Element dialog displays an ordered list of data elements comprising a composite data element, including:

Column Description
ID The data element number.
Description The user-defined description of the data element. By default, it is set to the standard X12 data element name.

Editing a Composite Data Element

To edit a composite data element:

  1. For the:
    • Description field, type directly into the field.
    • Structure of the composite data element, select the adjacent Browse icon to display the Composite Data Element dialog. To: 
      • Add a data element to the composite data element, select the data element to add from the drop-down list and select the Add button.
      • Delete a data element from the composite data element, select the data element to delete and select the Delete button.
      • Re-arrange the order of data elements in the composite data element, drag the data elements to the appropriate position.
  2. Select the OK button to save your changes, or the Cancel button to discard them.

Deleting a Composite Data Element

To delete a data element:

  1. Select the data element to delete.
  2. Select the Delete button.
  3. Select the OK button to save your changes, or the Cancel button to discard them.
