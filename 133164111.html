Starting up Rhapsody in 'Safe Mode'

Rhapsody incorporates a troubleshooting feature called 'safe mode'. When the Rhapsody Engine is started in 'safe mode', components configured to start automatically upon engine start-up are not started. Communication point scheduling is also disabled when this mode is active.

To start the engine in 'safe mode':

  1. Set the property StartupService.safeMode to true in the rhapsody.properties file (by default it is commented out and set to false).
  2. Restart Rhapsody in order for it to pick up changes in the rhapsody.properties file.
