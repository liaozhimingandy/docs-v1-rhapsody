FreeMarker Template Editor

The FreeMarker Template Editor enables you to create and manage FreeMarker template files used for configuring the FreeMarker Mapper filter.

The FreeMarker Template Editor displays the following information:

Panel Description
FreeMarker Template File panel Displays the FreeMarker template files in separate tabs
FreeMarker Toolbox panel

Displays the following tabs whose items you can drag to add to the currently selected FreeMarker template file:

  • Definition - displays the selected message definition.
  • Rhapsody - displays a list of FreeMarker API objects.
  • Includes - displays a list of open files in the FreeMarker Template File panel that you can add as include files.

You can perform the following actions in the FreeMarker Template Editor.

Action Description
New Enables you to create a new FreeMarker template file.
Open Enables you to browse and open an existing FreeMarker template file or an include file.
Save Saves any changes to the selected file.
Set as Configuration File Sets the selected FreeMarker template file to be the configuration file for the FreeMarker Mapper filter.
Revert Reverts the content of the selected file to the state it was in the last time it was checked in.
Select Definition Enables you to select a message definition.

Creating a New Template File

By default, the main FreeMarker template file is displayed in a tab in the FreeMarker Template File panel. You can begin to edit it directly.

To create a new auxiliary FreeMarker template file:

  1. Select the New button to display the New File dialog.
  2. Enter the filename of the FreeMarker template file (including the file extension), and select the OK button to create the template file.
  3. The new FreeMarker template file is displayed in a new tab in the FreeMarker Template File panel, and is ready to edit.

Selecting a Definition

To select a message definition:

  1. Select the Select Definition button to display the Select a message definition dialog.
  2. Select the definition, and select the OK button to add it to your file.

If you have selected an EDI HL7v2 message definition, you can drag the following elements of the message definition into your FreeMarker template file from a drop-down list in the Definition tab:

  • None (the element itself)
  • Get Element Value
  • Get Element
  • Get All Children
  • Get Element Index
  • Get Element Count
  • Is Element Empty
