My Leave (Do not send me notifications)

You can identify and enter days when you are not working and do not wish to receive 'Notification Messages'.

Field

Description

Start Date

Select the Calender icon and select the date when your leave begins.

End Date

Select the Calender icon and select the date when your leave ends.

Redirect Notifications To

Select E-mail, User, or No Redirection from the drop-down list and enter the appropriate text in the search field.
  • To add an email address to redirect notifications to, enter the email address in the text box and select the Add icon .
  • To add a user to redirect notifications to, enter the name in the text box and select the Add icon , or select the Users icon and select users from the Select User dialog.
  • Select the Remove icon to remove a leave.

If a user has sufficient access rights (admin rights), they can view My Leave (Do not send me notifications) settings for other users, enabling them to define leave for users on sick leave and redirect notifications accordingly.
