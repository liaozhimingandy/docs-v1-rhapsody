Changing Your Password

You can change your password at any time by selecting Rhapsody>Change Password... from the menu. This displays the Change Password dialog.

Passwords are case-sensitive. If a password policy has been defined, all password policy rules are displayed on the Change Password dialog.

Refer to Access Control in Rhapsody for details about resetting the password for another user or for the administrator.
