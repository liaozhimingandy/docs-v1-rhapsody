Uninstalling the Rhapsody Engine on Windows

It is recommended you stop the Rhapsody engine and the Rhapsody Service Monitor before beginning the uninstall process.

To uninstall the Rhapsody engine:

  1. Decide how you want to proceed:

    If you want to use the...

    Then...

    Command prompt

    Type <Rhapsody server>\Program Files\Rhapsody\Rhapsody Engine 6\uninstall.exe, and press ENTER.

    Windows® Control Panel

    a. Navigate to Start>Control Panel>Add or Remove Programs. Note that this path may differ depending on the version of Windows® you are using.
    b. From the program list, select Rhapsody <6.x.x>, then select the Change/Remove button.

  2. Select the Next button to start uninstalling:

  3. Select the additional tasks you want Rhapsody to carry out during the uninstallation process, then select the Next button.
    • Delete Rhapsody Data - selecting this checkbox deletes all Rhapsody data, including configuration data.

    • Delete Log Files - selecting this checkbox deletes all the Rhapsody engine logs.

  4. When Rhapsody engine has been uninstalled, a success dialog is displayed:

  5. Select the Exit button to exit the wizard.
