Creating the Client Route

Click the New Route icon on the standard toolbar, and call it Client Route. Add the PatientLookup.s3d definition to the route.

Complete the following:

  • Create a JavaScript Filter to Pick the Operation
  • Configure the Client Route

Create a JavaScript Filter to Pick the Operation

From the Filter Toolbox (Common menu), create a JavaScript filter called PickOperation.

var next = output.append(input[0]);
if (next.getField("Query/Type") == "ID") {
	next.setProperty("rhapsody:consumerOperation", "lookupById");
} else {
	next.setProperty("rhapsody:consumerOperation", "lookupByName");
}
