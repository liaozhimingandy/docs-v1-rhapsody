Base64 Encoding

The Base64 Encoding filter encodes and decodes between binary data and Base64 encoded data. Base 64 is a data encoding scheme whereby binary-encoded data is converted to printable ASCII characters.

When the Base64 Encoding filter is used to decode a GZIP Base64 encoded string, the filter automatically decompresses (unzips) the string in addition to Base64 decoding.

Refer to Encoding and Decoding Base64 Messages for scenarios using this filter.

Configuration Properties

Property

Description

Mode

Specifies whether messages should be encoded to or decoded from Base64:

  • Encode (default).
  • Decode.

Output Format

Specifies how the encoded data should be output:

  • Multiple lines (default) - as per the standard Base64 format (76 characters per line, separated by a carriage return-line feed pair).
  • Single line - as a single line of encoded characters with all carriage return-line feed pairs removed.
