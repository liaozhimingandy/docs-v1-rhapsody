Managing Filters

To manage filters, refer to:

  • Adding a Filter to a Route
  • Configuring a Filter
  • Deleting a Filter
  • Creating a Copy of a Filter

Adding a Filter to a Route

 To add a filter to a route:

  1. From the Filters tab in the Filter Toolbox, if required, search for the filter you want to add by typing its name into the search box.

  2. Drag a filter, and drop it onto the route canvas in the Route Editor.

    If a template has been set as the default template for the selected filter type, then the filter is created using the default template's configuration settings. If no template for the selected filter type has been set to default, then the filter is created using the default Rhapsody configuration settings.

Alternatively:

  1. From the Templates tab in the Filter Toolbox, if required, search for the filter template you want to add by typing its name into the search box.
  2. Drag a filter template, and drop it onto the route canvas in the Route Editor.

If the filter is dropped on top of an existing connector, the filter will be inserted at that point in the route pathway. If the connector is not highlighted as the filter is dropped, the filter will simply be placed onto the canvas but will not be joined to the connector. A filter can be dropped onto a blank area of the canvas and connections made to and from it by selecting the Connector iconand dragging the mouse pointer between components.

Selected communication points and filters are highlighted by a blue rectangle. The direction of the connection is indicated by an arrowhead on the connector.

Configuring a Filter

To configure a filter's properties, open the filter's Properties dialog by using one of the following methods:

  • Double-click on the filter in the Route Editor.
  • Right-click on the filter in the Route Editor and select Properties....
  • Right-click on the filter in the Workspace and select Properties....
  • Right-click on the filterin the Object Browser and select the Modify Properties button.

Refer to Configuring Filters for details on the filter properties you can configure.

Deleting a Filter

To remove a filter from a route:

  1. Select the filter in the Route Editor.
  2. From the right-click menu, select Delete Component (or press Del).
  3. Select the Yes button in the confirmation dialog to remove the filter, or the Cancel button to abort.

Creating a Copy of a Filter

To create an exact copy of a filter, including its configuration:

  1. Select the filter to copy in the Route Editor.
  2. From the right-click menu, select Copy (or press Ctrl+C).
  3. To copy the filter:
    1.  to the same route, select Paste (or press Ctrl+V) in the Route Editor for the same route.
    2. To a different route, right-click on a route in the Workspace you want to copy the filter to, and select Paste (or press Ctrl+V).

To copy the filter to the same route, you can also press the Ctrl key on your keyboard, and drag-and-drop the filter to a new location in the Route Editor for the same route.
