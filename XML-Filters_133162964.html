XML Filters

XML filters process messages in XML format, performing the following tasks:

  • Web Services Signing
  • XML Encryption / Decryption
  • XML Signing / Verifying
  • XML to PDF
  • XML to RTF
  • XML Validation
  • XSLT Stylesheet
  • Zip XSD Validation
