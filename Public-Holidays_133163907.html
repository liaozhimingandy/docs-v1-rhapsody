Public Holidays

Menu Path : Notifications>Public Holidays

Write Access Rights:

  • Log in to Management Console
  • Edit public holidays


The Public Holidays page enables you to specify days public holidays occur. All public holidays entered on this page are included when a user is scheduled to receive notifications for a watchlist.

The Public Holidays page is only available to administrators who have access to view the page.

 

Column

Description

Holiday

The name of the holiday.

Date

The date when the holiday occurs.

Export

Enables you to export holidays saved on the Public Holidays page either for a backup or editing purposes. The export file name is publicholidays.xml and can be saved on your local machine.

If required, the exported file can be edited and then imported into Rhapsody. Following is the format of the exported file:

<PublicHolidays xmlns="http://www.orionhealth.com/rhapsody/2010/12/notifications/publicHolidays">
  <PublicHoliday name="Christmas Day" date="2011-12-25"/>
  <PublicHoliday name="Boxing Day" date="2011-12-26"/>
</PublicHolidays>

Import

Enables you to import a previously saved or new publicholidays.xml file.

Adding a Holiday

To add a holiday:

  1. Enter the holiday name in the Holiday column text box, click the Calendar icon and select a date the holiday falls on.
  2. Select the Add icon . The holiday is added to the table.
  3. Select the Apply button to save the changes made to the page.

Removing a Holiday

To remove a holiday, select the Remove icon  associated with the holiday you want to delete.
