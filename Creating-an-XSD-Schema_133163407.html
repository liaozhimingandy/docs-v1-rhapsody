Creating an XSD Schema

The XSD Generator takes an XML document as input and generates an XSD schema based on the XML document. This utility is extremely useful when using a Rhapsody component that generates an XML document based on the configuration of the component; for example, the database communication point.

To create a new XSD schema:

  1. Select the XSD Generator command from the Tools drop-down menu. The Rhapsody IDE XSD Generator is displayed:

  2. Click the Load XML Documents link to display the WindowsÂ® Open dialog.

  3. Browse to and select the .xml from which you want to create the XSD schema and then select the Open button.

    One or more .xml files can be loaded and used to generate the XSD schema.

    The XSD Generator then loads the .xml file.

  4. Click the Generate XSD Schema link to generate the XSD schema from the .xml file. The XSD Generator then generates the XSD schema from the .xml file:

  5. Click the Save XSD Schema link to save the XSD schema to the required location.
