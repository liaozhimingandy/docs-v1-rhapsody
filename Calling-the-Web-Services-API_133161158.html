Calling the Web Services API

This section provides examples of how to call the Rhapsody Web Services API.

  • Calling the Web Services API using SoapUI provides an example of how to call the API using the free SoapUI tool.
  • Calling the Web Services API using .NET provides an example of how to call the API from a .NET project.
  • Calling the Web Services API using a Communication Point provides an example of how to configure the Web Service Client communication point to call the API to start and stop a component.
