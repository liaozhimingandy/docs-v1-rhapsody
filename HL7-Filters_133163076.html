HL7 Filters

Rhapsody provides the following HL7-related filters:

  • FHIR XML / JSON
  • HL7 Acknowledgement Generation
  • HL7 Code Translation and Validation Filters
  • HL7 Detect Character Encoding
  • HL7 Fill Empty Fields
  • HL7 Message Modifier
  • Intelligent Mapper (Filter)
  • Date Validation
