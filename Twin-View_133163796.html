Twin View

The twin view enables you to:

  • Understand how a message changes on the message path.
  • Compare the message bodies and properties of related messages.

To access the twin view, click the  icon for the message and select Compare In and Out from the Event Actions dialog. Twin view is displayed in both flatwire and tree mode. 

Scenarios

The twin view can be activated for the following scenarios:

  • Input and Output Message
  • Tracked Original and Tracked Response Message
  • Tracked Response and Tracked Original Message
  • Redirected Response and Redirected Original message
  • Output and Response Message
  • Response and Output Message
  • Appended and View Message
  • View and Merge Message

Input and Output Message

The Input message displays the message at the selected node, while the Output message displays the message as it exits the node in the message path:

Tracked Original and Tracked Response Message

The Tracked Original message is the message leaving a Rhapsody component that has message tracking configured. The Tracked Response message is the response from the external system. Occurs when you are viewing a tracked message and select a response message identifier from the message event description.

Tracked Response and Tracked Original Message

Similar to Tracked Original and Tracked Response Message, except you are viewing the response message and selecting an original message identifier from the message event description.

Redirected Response and Redirected Original message

When a message is redirected, a new instance of the message is created to redirect to the selected component. The root node on the message path of the redirected message becomes an archived message (the Page iconindicates the message is an archived message.) The left-hand side displays the redirected message and the right-hand side displays the redirected archive message (original message).

Output and Response Message

When a communication point is configured in Out-In mode, the output message is tracked with out-in tracking. The response input message is linked to the output message and is displayed in the message event description.

Response and Output Message

Similar to Output and Response Message but the viewing order is reversed. Occurs when you are viewing the response message and select the matched output message from the message event description.

Appended and View Message

When multiple incoming Rhapsody messages are merged into one (such as in the case of batched messages), the message event contains more than one linked message identifier. Click one of the links in the Description column, to display the merged message and view message in twin view.

View and Appended Message

Similar to Appended and View Message but the viewing order is reversed.
