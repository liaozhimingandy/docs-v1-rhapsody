Unattended Installations

This section describes how to:

  • Install the Rhapsody engine and Rhapsody IDE in silent or unattended mode.
  • Configure Rhapsody IDE unattended.

Registered users can download the Rhapsody installation files from the Rhapsody FTP site.

We recommend that you close (quit) all programs before you begin.

Rhapsody Engine and Rhapsody IDE can be installed in any order.
