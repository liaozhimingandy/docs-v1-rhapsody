Web Services API

The Rhapsody Integration Engine supports the following web services:

Service

Provides

Rhapsody Status Web Service

Simple ping and status operations.

Rhapsody Components Web Service

Operations to start and stop components, and retrieve a list of components and simple component information.

System Statistics Web Service

Operations to retrieve message counts and memory statistics, and the ability to export those statistics.

Message Searching Web Service

Operations to view messages, find them on the Error and Hold Queues, as well as release, reprocess and delete them from the Error and Hold Queues.

Lookup Table Web Service

Operations to lookup values contained within Rhapsody lookup tables.
