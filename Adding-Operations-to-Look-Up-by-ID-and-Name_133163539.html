Adding Operations to Look Up by ID and Name

Complete the following:

  • Add Operation - lookupById
    • Input Tab
    • Output Tab
    • Faults Tab
  • Add Operation - lookupByName
    • Input Tab
    • Output Tab
    • Faults Tab
  • Check In the Web Service

Add Operation - lookupById

Go back to the General tab to add the first operation. Under Operations, click the Add button.

Refer to Adding and Editing Operations for field definitions.

Field

Description

Name

lookupById

Description

Looks up a patient by their identifier

Operation Mode

Request-Response

SOAP Action

urn:lookupById

Field

Description

Check In the Web Service

On the Web Services screen, select PatientLookupService, then click the Check In button. The state of the service should change to Running.
