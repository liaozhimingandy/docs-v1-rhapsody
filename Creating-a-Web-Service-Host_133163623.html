Creating a Web Service Host

Click the New Communication Point icon on the standard toolbar to create a new Web Service Hosting communication point called HL7 Message Acknowledgement Server Web Service Hosting.

Complete the following:

  • General Tab
  • Configuration Tab
    • Add Operation - AcknowledgeHL7Message
      • Operation Parameter Mapping - Input Tab
      • Operation Parameter Mapping - Output Tab

General Tab

Field

Description

Connection Mode

In->Out

Configuration Tab

Click the Edit Communication Point Configuration link.

Field

Description

Web Service

Select HL7AcknowledgementService

Refer to Web Service Hosting Communication Point for field definitions.

Add Operation - AcknowledgeHL7Message

  1. Click the Add Operation button.
  2. Select AcknowledgeHL7Message (Request-Response), then click the OK button.
