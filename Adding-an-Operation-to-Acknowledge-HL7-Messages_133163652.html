Adding an Operation to Acknowledge HL7 Messages

Complete the following:

  • Add Operation - AcknowledgeHL7Message
    • Input Tab
    • Output Tab
    • Faults Tab
  • Check In the Web Service

Add Operation - AcknowledgeHL7Message

Navigate to the Operations tab to add the first operation. On the Operations tab, click the Add Operation button

Refer to Adding and Editing Operations for field definitions.

Field

Description

Name

AcknowledgeHL7Message

Description

Returns an acknowledgement on successfully parsing an HL7 message

Operation Mode

Request-Response

SOAP Action

urn:AcknowledgeHL7Message

Input Tab

Field

Description

Parameter Type

 

Use existing type

<Any Type>

 

Output Tab

Field

Description

Parameter Type

 

Use existing type

xs:string

 

Faults Tab

Field

Description

Parameter Type

 

Use existing type

xs:string

HL7AcknowledgementService should now include the AcknowledgeHL7Message operation.

Check In the Web Service

On the Web Services screen, select HL7AcknowledgementService, then click the Check In button. The state of the service should change to Running.
