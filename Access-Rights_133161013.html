Access Rights

The Rights Viewer dialog enables you to view the following access rights, which are grouped according to function:

  • Global Access Rights
    • Login
    • Preferences
    • Rhapsody IDE
    • Management
    • Certificates and Keys
    • Rhapsody Variables
    • Shared JavaScript Libraries
    • Web Services
    • Notifications
    • Searches
    • Logs
    • Statistics
    • JavaScript Counters
    • Lookup Tables
    • REST API
    • Administration REST API
  • Locker Access Rights
    • Rhapsody IDE
    • Management
    • Communication Points and Routes
    • REST Clients
    • Intelligent Mapper
    • Searches
    • Messages
    • Database Input-Key Values
    • REST API

The internal value for access rights is used by the REST API.
