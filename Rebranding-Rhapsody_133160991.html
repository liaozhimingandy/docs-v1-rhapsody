Rebranding Rhapsody

This section contains information for Rhapsody® partners. It is confidential, and is not part of the standard Rhapsody Integration Engine (Rhapsody) documentation.

Rhapsody® partners and resellers can insert their own logos and branding into Rhapsody IDE and the Management Console. You must have suitable images of a defined size to replace the Rhapsody brand images. For more details, refer to:

  • Rebranding Rhapsody IDE
  • Rebranding the Management Console

The Rhapsody installation is not configurable, but may be run in Silent mode. This allows partners to install Rhapsody, and then install the required configuration options to insert their branding before users run the applications.
