The Rhapsody Platform

The Rhapsody Integration Engine (Rhapsody) is a powerful, easy-to-use integration engine that enables you to integrate and communicate between various (often incompatible) systems. Rhapsody manages and streamlines message exchange between applications, databases and external systems.

Messages enter the Rhapsody engine via input communication points and pass through processing filters before exiting the engine via output communication points. The Rhapsody engine analyzes the incoming messages to determine which steps, procedures or filters need to be applied in handling the message and passing it to its destination.

The Rhapsody engine is configured using the Rhapsody Integrated Development Environment (Rhapsody IDE), a Windows-based application, and monitored through the web-based Management Console and Rhapsody Dashboard. The engine also provides a REST API for remote monitoring and administration.

EDI and XML message designer tools in the Rhapsody Messaging Toolkit are used to create message definition files that inform Rhapsody of the structure of the messages it is processing. Custom-written filters and communication points can be easily created and dynamically loaded to augment the engine's capabilities.
