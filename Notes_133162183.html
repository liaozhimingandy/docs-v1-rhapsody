Notes

The Notes tab allows you to attach free-form Rich Text Format notes to a communication point. These notes are not used by the engine but appear in any generated documentation. The text can be formatted using the formatting toolbar.

To use notes as support notes for display in the Management Console, check the Use notes as support notes checkbox.
