Using SSL Certificates

The Rhapsody Integration Engine uses SSL to secure the communications to any connected Rhapsody IDEs, and to any web browsers connected to the Management Console. When the SSL (or HTTPS) connection is established, the Rhapsody engine presents an SSL certificate that can be used to verify its identity. This is important so that a Rhapsody IDE or web browser connecting to the Rhapsody engine can verify the engine's identity before it transmits the login username and password.
