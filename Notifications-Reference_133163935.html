Notification events in the WindowsÂ® Event Viewer as displayed as an EventID and Qualifier:

  • The EventID is the equal as the Code assigned to the event.

  • The Qualifier can be obtained from the formula (Windows Event ID - Code) / 65536.

For example, for the event 'Route Started', the Windows Event ID = 1090715650, and Code = 2. Therefore the EventID is 2, and the Qualifier is (1090715650 - 2) / 65536 = 16643.
