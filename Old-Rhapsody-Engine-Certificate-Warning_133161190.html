Old Rhapsody Engine Certificate Warning

The Old Rhapsody Engine Certificate warning dialog is displayed when connecting to a version of the Rhapsody engine that is using a shared SSL certificate, rather than a unique SSL certificate generated explicitly for that engine. We strongly recommend the Rhapsody engine be upgraded to the latest service pack available for that version as soon as possible to allow a unique SSL certificate to be used for the IDE connection.

Action Description
No Cancels the connection so the login does not take place.
Yes Allows the login attempt to proceed. Note that there is no mechanism to permanently trust this connection, and so this warning is shown again the next time a connection is made to this Rhapsody engine.
