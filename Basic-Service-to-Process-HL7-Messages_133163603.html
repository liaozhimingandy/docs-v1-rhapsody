Basic Service to Process HL7 Messages

This tutorial will use a web service (hosted in Rhapsody) as the transport mechanism for HL7 2.x messages. The web service will acknowledge the HL7 message by returning an HL7 acknowledgment message as the web service response.

Before you add a Web service, you must set up the following:

  • Create the Folder Structure for the Communication Points and Routes
  • Load the Message Definition File

Create the Folder Structure for the Communication Points and Routes

In Rhapsody IDE, click the New Locker icon on the standard toolbar to create a new HL7WebService locker, and then click the New Folder icon on the standard toolbar to create two child folders â€” Client and Server.

Load the Message Definition File

View>File Manager...

Using the File Manager, load the message definition file HL7v2.3.1.s3d (download) into the HL7WebService locker.
