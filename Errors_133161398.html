Errors

Two dataMapExceptions are thrown:

  • dataMap failed to find 'input' in translation file 'fileName'. No default value supplied or the input column was missing from all rows.
    Occurs when the input value is not in the translation file, there is no %default% line, and the default value argument is "". Also occurs when all rows are missing the input column regardless of if a default was supplied or not.
  • Failed to read dataMap translation file: fileName.
    Occurs when the translation file cannot be found or an error occurs while reading the file.
    When either of the errors occur, the message is sent to the error queue or via the filter's error connector.
