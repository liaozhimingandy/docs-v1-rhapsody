Monitoring Lookup Tables

Menu Path : Management>Lookup Tables

Read Access Rights:

  • Log in to Management Console
  • View lookup table values

Write Access Rights:

  • Edit lookup table values

The Lookup Tables page lists the lookup tables available in Rhapsody. The following lookup table functions are supported in the Management Console:

  • Viewing and Updating Lookup Tables.
  • Importing and Exporting Lookup Tables.
  • Resolving Lookup Failures.

If you want to create and delete tables, or edit the metadata or structure of a lookup table, you must do so via Rhapsody IDE. Refer to Lookup Tables for details.




The lookup tables list auto-refreshes every five seconds.

Column

Description

Name

The name of the lookup table.

Description

A description of the lookup table.

Rows

Number of rows in the table.

Missing Values

The number of missing values detailed in the Lookup Failures table. Not recording is displayed when the recording of failures is turned off in Rhapsody IDE.

Failure Count

Total number of failures associated with the lookup table.

Last Modified

Date and time the table was last modified.

When you click on or hover over the action icon, a popover is displayed with four action links: Export Values, Export Failures, Import Values and Clear All Failures. Refer to Importing and Exporting Lookup Tables for details.

Export Failures or Clear All Failures options do not appear if the associated Failure Count is zero.
