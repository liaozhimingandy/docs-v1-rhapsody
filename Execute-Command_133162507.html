Execute Command

The Execute Command communication point enables you to specify and run an executable file for delivering a message.

Supported Operational Modes: Output

Output Properties

Property

Description

Command Line

The specific command required to run an executable file.

For example, in WindowsÂ®, you can use the following command to run the executable file filter.cmd located in the directory c:\test:

c:\test\filter.cmd

Working Directory

The directory where you want the executable file to run (this directory is not necessarily where the executable file is located).

Body as Parameter

Whether to append the body of the message to command line as a parameter:

  • Enabled - the body of the message is appended to the command line as a parameter.
  • Disabled (default) - the body of the message becomes the standard input (stdin) of the command line.
