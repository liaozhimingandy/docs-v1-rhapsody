GUI and Console Installations

This section describes how to:

  • Install the Rhapsody engine and Rhapsody IDE on a Windows® environment.
  • Install the Rhapsody engine on a *nix environment.
  • Uninstall Rhapsody.

Before You Begin

We recommend you complete the following before you install Rhapsody:

  • Check you meet the System Requirements.

  • Read the installation caveats.

  • Obtain the required installers from Rhapsody Support.

  • Ensure you have obtained the correct licenses.

  • Refer to Rhapsody Datastore Setup for important information on configuring your datastore.
  • Close (quit) all programs.

Instructions

For instructions on:

  • Installing Rhapsody IDE on a Windows® environment, refer to Installing Rhapsody IDE. The Rhapsody engine and Rhapsody IDE can be installed in any order.
  • Installing the Rhapsody engine and Rhapsody IDE on a Windows® environment, refer to Installing the Rhapsody Engine on Windows.
  • Installing the Rhapsody engine on a *nix environment, refer to Installing the Rhapsody Engine on *nix.
  • Uninstalling Rhapsody, refer to Uninstalling Rhapsody.
