Batching Messages

To create a batch of messages, the Mode configuration property must be set to Batch, and separator information must be defined in the Separator property, as shown in the following screenshot:

The default (separate messages) means that the separator character will be used between each message only, with no trailing separator at the end of the last message.

To define the size of the batch message created, Message Collector properties of the filter are used. The Message Collector properties allow the filter to wait for a set number of messages, or a set amount of time. Select the required option in the Message Collector Type field.
