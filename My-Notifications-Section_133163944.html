My Notifications Section

The My Notifications section displays a list of Watchlists you belong to and components you are monitoring and receiving notifications for.

Watchlists Table

Column Name

Description

Watchlist

Displays the names of all watchlists you belong to. Click on a watchlist to view its details.

Components Table

Column Name

Description

Component

Displays the names of individual components you are monitoring and receiving notifications for. Click on a component to view its details.

Notification Level

Displays icons that determine the notification level for a component.

The three icons show the notification levels enabled for a component. Clicking the icons toggles between the following three states:

  • Escalations only.
  • Escalations, and Alarms.
  • Escalations, Alarms, and Warnings.
  • Select the Add icon to add a component to the table.
  • Select the Remove icon to remove a component.
