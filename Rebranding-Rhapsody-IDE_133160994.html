Rebranding Rhapsody IDE

Custom Enhancements

Custom branding images are compiled into a .NET RESOURCES file and placed in the (<RhapsodyInstallDirectory>\Rhapsody IDE\Rhapsody IDE) directory. A command line tool (RhapsodyBrandCompiler.exe) is used to generate the RESOURCES file, which takes an XML configuration file as its parameter.

The following locations can be rebranded in Rhapsody IDE:

  • Login Screen
  • Splash Screen
  • 'About Rhapsody' Dialog
  • Change Password Form

To learn how to apply custom branding to Rhapsody IDE, refer to Customizing Rhapsody IDE Screens.

Customizing Rhapsody IDE Screens

  1. Download the Rhapsody Brand Compiler tool to a local directory.

  2. Place your XML configuration file in the same directory. The file name is not important, but it must have a .xml extension. The XML configuration file contains details of the resources you want to compile:

    XML Configuration File 
    <BrandingConfiguration>
    <AboutLogo filename="About.png"/>
    <LoginBanner filename="Login.png" textCol="255,255,255"/>
    <SplashScreen filename="Splash.png" textCol="255,255,255" statusTextCol="0,0,0"/>
</BrandingConfiguration>
  3. Place your image files in the same directory.
  4. Open a command prompt, and enter: RhapsodyBrandCompiler.exe <XMLConfigurationFile>.xml. This generates a .resources file with the same name as the XML file. For example, if your XML file is called config.xml, the RESOURCES file will be called config.resources.
  5. Copy the.resources file to the <RhapsodyIDEInstallDirectory>\Rhapsody IDE 6\Rhapsody IDE directory.
  6. Start Rhapsody IDE to view your changes. Rhapsody IDE will load the first .resources file that it finds, otherwise it will use the default Rhapsody images from the embedded resources.

Images can be of type .png, .bmp, .jpg, .tiff or .gif. Your custom image is scaled to fit if it is larger or smaller than the specified dimensions.

The images on the Login and Splash screens are overlaid with product information text. The RESOURCES file also contains the text colors for these screens specified as an RGB string, for example: 255, 255, 255.
