File Menu

Using the File menu, you can load and save Rhapsody configurations:

Option

Icon

 Keyboard Shortcut

Description

Open...

CTRL+O

Load a new Rhapsody configuration onto your engine. Refer to Loading a Configuration for details.

Save...

CTRL+S

Save your current Rhapsody configuration as a single configuration file (RLC), which can then be imported into a second Rhapsody engine. Refer to Saving a Configuration for details.
