Archiving

Archive Message Bodies

Applies to...
  • Operational Mode: Input, Out->In.
  • Connection Type: FTP, FTPS, SFTP.

Archive Message Bodies indicates whether messages received by the (S)FTP communication point are to be archived as per the standard archiving strategy or short-term archiving strategy. Refer to:

  • Message Store for details on how messages are stored.
  • Archive Cleanup for details on how archived messages are cleaned up.
Property Option Description
Normal Archiving Strategy† Messages received are retained for the user-specified archive period, after which time they are deleted.
Cleanup After Sending Message

Messages are marked for clean up when they leave Rhapsody and their message body is deleted the next time archive cleanup runs.

† Default option.
