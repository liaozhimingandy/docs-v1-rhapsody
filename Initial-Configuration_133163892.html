Initial Configuration

Notification Delivery Methods

To configure Rhapsody to use a mail server:

  1. In the Management Console, navigate to Notifications>Delivery Methods.

  2. Select the Enable notifications to be sent via email checkbox and enter the following details:

    Field

    Description

    Server

    Enter the mail server to send email through.

    Port

    Enter the mail server port: 25 for SMTP and SMTP with TLS; 465 for SMTPS.

    Protocol

    Select the encrypted connection from the drop-down list. Options are SMTP (plain Text), SMTPS (SMTP over SSL) and SMTP with TLS encryption (SMTP with StartTLS).

    Authentication

    Use authentication if the mail server requires a username and password to send email.

    • If authentication is not required, click the No button.
    • If authentication is required, click the Yes button and enter the Login name and Login Password in the text fields.

    From Address

    Enter the host email address from where notification emails will be sent.

    Flood Control

    Enter the maximum number of notifications to send, in a five-minute window, before switching into batch notification mode.

    Test Email Address

    Enter an email address in the text box and click the Send Test Email link to test if a notification is sent to the entered email address and to ensure the Rhapsody Engine can send alerts.

  3. Click the Apply button to save the changes made to the page.

    If your organization supports sending SMS or Pager notifications, you can configure the settings for those on this page too.

User Notification Addresses

The first time a user tries to set up a notification of any kind, they need to configure the addresses that notifications can be sent to.

To configure the address:

  1. In the Management Console, navigate to Notifications>My Notifications.

  2. In the Send Notifications To panel, enter or update (as required) your email address.

    If you wish to receive SMS or Pager notifications (and your organization has appropriate facilities for this), you can enable and configure your SMS and pager addresses here too.

  3. Click the Apply button to save the changes made to the page.
