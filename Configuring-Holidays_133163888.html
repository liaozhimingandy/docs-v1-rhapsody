Configuring Holidays

To define public holidays:

  1. Log into the Rhapsody Management Console and navigate to Notifications>Public Holidays.
  2. Enter the holiday name in the Holiday column text box, select the Calendar icon and select a date the holiday falls on.
  3. Select the Add icon . The holiday is added to the table. Select the Remove icon to remove a holiday.
  4. Select the Apply button to save the changes made to the page.
