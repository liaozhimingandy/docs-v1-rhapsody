Concatenation

You can concatenate values in a message. For example, you can set a field to a series of field values concatenated using a separator.

To concatenate a series of auxiliary operands and set them to a target operand:

  1. Select the target operand.
  2. Select the By Concatenating action operator to display the Concatenation form:
  3. Enter the auxiliary operands.
  4. Enter the concatenation separator into the Separator Between Values field (or leave blank if you want no separator between the auxiliary operands).
  5. Click the OK link to save your changes, or the Cancel link to discard them.
