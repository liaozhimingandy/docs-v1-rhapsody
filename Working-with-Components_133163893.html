Working with Components

How to Receive Notifications from a Specific Route or Communication Point

  1. Log into the  Management Console and navigate to Monitoring>Communication Points or Monitoring>Routes.
  2. Select the Communication Points/Routes that you wish to receive notifications for.
  3. Click the Add Notifications link at the bottom of the page.
  4. From the Add Notifications lightbox, select Person (24/7) from the Send To drop-down list, search for a user and click the Add icon .
  5. Review the Resend and Escalation Periods as appropriate.
  6. Select the Apply button to save your changes.

How to See the Components You Will Receive Notifications For

Log into the  Management Console and navigate to Notifications>My Notifications.

  • You will see all the Components you are explicitly receiving notifications for in the Components table at the bottom of the page.
  • If you know the name of the Component you wish to add, you can use the search box to search for a component and click the Add icon .

On this page, you will also see Watchlists you are subscribed to. To view the Components that you will receive notifications for via the Watchlist, you must view the details for the Watchlist by clicking on the Watchlist name.

How to Enter Leave to Stop Receiving Notifications

To schedule a period of time to stop receiving notifications:

  1. Log into the  Management Console and navigate to Notifications>My Notifications.
  2. In the My Leave section, enter a Start Date and End Date.
  3. If you wish to redirect your notifications during this time, set the Redirect Notifications To drop down list either to E-Mail or User and use the text box to enter an e-mail address or search for a user. If you do not wish to redirect the notifications, choose No Redirection.
  4. Select the Add icon .

How to See the Components on a Watchlist

  1. Log into the Rhapsody Management Console and navigate to Notifications>Watchlists.
  2. Select the Watchlist from the list.
  3. Following details will be displayed:
    • A list of users that are receiving notifications from this Watchlist.
    • A list of the users that have Leave specified and for when.
    • A list of Components that are on this Watchlist.
    • A list of Web Services that are on this Watchlist.

How to See the Watchlists a Component Is On

  1. Log into the Rhapsody Management Console and navigate to Monitoring>Communication Points or Monitoring>Routes.
  2. Select the component you wish to view.
  3. The Notifications pane will display a list of People and Watchlists that are receiving Notifications from this Component.

How to Suspend and Resume all Notifications from a Component

  1. Log into the Rhapsody Management Console and navigate to Monitoring>Communication Points or Monitoring>Routes.
  2. Select the component you wish to view.
  3. On the Notifications page, click the Suspend All link to suspend all notifications for the selected component. When you suspend notifications for the selected component an information banner is displayed above the Activity Feed.
    Click the Resume Notifications link to resume notifications for the selected component. Refer to Suspend / Resume All for details.

To Suspend ALL notifications in Rhapsody:

  1. Log into the Rhapsody Management Console and navigate to Notifications>Suspend / Resume All.
  2. Enter a time (in minutes) in the Suspend for field and click the Suspend Now button. A confirmation dialog is displayed.
  3. Select the Suspend Now button. Notification messages for all components and the system will commence at the end of the time specified. Alternatively, click the Resume Notifications button to commence the sending of notification messages.
