Creating a Web Service Client

Click the New Communication Point icon on the standard toolbar to create a new Web Service Client communication point called HL7 Message Acknowledgement Web Service Client.

Complete the following:

  • General Tab
  • Configuration Tab
    • Web Service Consumer Communication Properties: General Tab
    • Web Service Consumer Communication Properties: Security Tab
    • Web Service Consumer Communication Properties: Operations Tab
      • Operation Editor: Request Tab
      • Operation Editor: Response Tab
      • Operation Editor: Fault Tab
