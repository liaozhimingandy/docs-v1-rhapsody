Find Messages

Find Messages provides search functions, which enable you to search for messages sent and received by Rhapsody:

  • Input Search
  • Output Search
  • Route Search
  • Message Collector Search
  • Saved Searches
  • Recent Results

Rhapsody supports indexed properties to allow for fast lookups of message content extracted as properties. These can be configured on a per route basis and can also be configured using JavaScript calls. These message properties appear as separate options under the Advanced Search Options for each type of search.
