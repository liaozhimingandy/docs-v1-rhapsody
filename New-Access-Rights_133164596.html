New Access Rights

Users, who are already familiar with Rhapsody, sometimes encounter unexpected behavior when trying to access new features in a new version of Rhapsody. The purpose of this section is to familiarize users with the new access rights added in successive releases beginning with Rhapsody 6.2:

  • Rhapsody 6.5
  • Rhapsody 6.4
  • Rhapsody 6.3
  • Rhapsody 6.2.5
  • Rhapsody 6.2.4
  • Rhapsody 6.2.3
  • Rhapsody 6.2.2
  • Rhapsody 6.2.1
  • Rhapsody 6.2

As in previous releases, after being granted access rights, users must start a new Rhapsody IDE session (log out and log back in) in order for the changes to take effect.

Rhapsody 6.5

Access Right

Description
Export logs REST API Required to call the REST API to export logs.
Manage product usage Required to manage the product usage configuration.
Manage product usage REST API Required to call the REST API to manage product usage settings.
Test filter and conditional connector REST API Required to call the REST API to manage filter and conditional connector tests.
Perform full text searches Required to perform full-text search for messages from all search pages in the Management Console.
For a full list of access rights for Rhapsody 6.5, refer to  Access Rights.

Rhapsody 6.4

Access Right

Description
View message in an emergency Required to view the contents and properties of a message on the Message View page in an emergency situation.
Edit templates Required to add, edit or delete templates that are available across Rhapsody.
Save templates REST API Required to call the REST API to save templates.
Load templates REST API Required to call the REST API to load templates.
View support notes REST API Required to call the REST API to view support notes of communication points, routes, and filters.
Stop engine REST API Required to call the REST API to stop the engine.
Suspend/resume global notifications REST API Required to call the REST API to suspend and resume all component and issue notifications across the engine.
Suspend/resume component and issue notifications REST API Required to call the REST API to suspend and resume the notifications of components and issues
Edit unmonitored component notifications REST API Required to call the REST API to edit notifications for unmonitored components.
Manage general settings REST API Required to call the REST API to change the Management Console's general settings.

Rhapsody 6.2.3

Access Right

Description

Export RPG private keys

Required to export RPG private keyrings.

Change system notifications REST API

Required to call the REST API to change system notification alerts.

Organize archive cleanup REST API

Required to call the REST API to manage archive cleanup.

Organize message defragmentation REST API

Required to call the REST API to manage message defragmentation.

Get backup information REST API

Required to call the REST API to fetch information about engine backups.

Run Intelligent Mapper tests

Required to run tests on a mapping project in the Intelligent Mapper.

Rhapsody 6.2.2

Access Right

Description

Start/stop communication points with scheduling enabled

Required to start and stop scheduled communication points at the locker level in order to minimize inadvertent start-up.

Run Intelligent Mapper analysis

Required in order to analyze messages in the Intelligent Mapper. This access right is not granted by default to users.

Read transient state REST API

Required in order to call the REST API to read the transient state.

Modify transient state REST API

Required in order to call the REST API to modify the transient state.

Rhapsody 6.2.1

Rhapsody 6.2.1 introduces no new access rights.

Rhapsody 6.2

Access Right

Description

Organize message defragmentation 

Required for editing the message defragmentation settings on the Archive Cleanup page, and calling the REST API to manage message defragmentation.

View notification settings REST API

Required for calling the REST API to view the system notifications settings.

View engine statistics REST API

Required for calling the REST API to read the in-use memory, memory used by the JVM, and total allocated memory in the engine. This access right has replaced ‘View memory statistics REST API’.

Save access groups REST API

Required for calling the REST API to save access groups.

Load access groups REST API

Required for calling the REST API to load access groups.

View support Notes

Required for viewing support notes for components in Rhapsody IDE and the Management Console.
