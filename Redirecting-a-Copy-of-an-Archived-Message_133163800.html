Redirecting a Copy of an Archived Message

From the Message View page, you can edit and redirect a copy of an archived Rhapsody message to another Rhapsody component.

To edit and redirect a copy of an archived message:

  1. On the Message View page, click the Redirect Edited Copy button. The Edit Message dialog is displayed:

  2. Edit the message body, then click the Apply button. The Select the Redirect Target Component dialog is displayed:

  3. Select the component you want to redirect the message to. The Message View page is displayed.
