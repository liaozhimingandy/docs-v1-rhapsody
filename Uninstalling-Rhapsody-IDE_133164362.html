Uninstalling Rhapsody IDE

To uninstall the Rhapsody™ Integrated Development Environment (Rhapsody IDE):

  1. Decide how you want to proceed:

    If you want to use the...

    Then...

    Command prompt

    Type <Rhapsody IDE installation directory>\uninst.exe, and press ENTER.

    Windows® Control Panel

    a. Click Start>Control Panel>Add or Remove Programs. Note that this path may differ slightly depending on the version of Windows® you are using.
    b. From the program list, select Rhapsody IDE 6, then click the Change/Remove button.

  2. Select the Yes button on the confirmation dialog to confirm that you want to uninstall Rhapsody IDE. The Rhapsody IDE Uninstall dialog is displayed:

  3. Select the Uninstall button. As the Rhapsody components are removed, a progress bar is displayed. Dialogs may also be displayed as Active-X Objects are identified for removal and unregistered. 

  4. When the uninstaller successfully completes uninstalling Rhapsody IDE, a success dialog is displayed. Select the OK button to close the dialog and exit.

Shared and Locked Files

While uninstalling, the wizard may identify some files that are:

  • Shared with other applications – note the location of the shared file. If it is in a Rhapsody directory, you may delete it safely.
  • Locked by other applications – the locked file can be left on the hard drive, or removed by rebooting the machine.
