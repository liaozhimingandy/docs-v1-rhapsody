Monitoring System Configuration

To receive system notifications:

  1. In the Management Console, navigate to Monitoring>System Monitor.
  2. In the Notifications section, click the Change link.
  3. Add the user, email address or Watchlist as a recipient.
  4. Click the Apply button to save the changes made to the page.
