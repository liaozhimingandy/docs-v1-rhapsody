Message Tracking in the Management Console

The Management Console enables you to monitor tracked messages while it is running. Failed messages can be followed up, and the tracking events can be viewed for each message. From the:

  • Output Search page, you can search for messages based on Tracking State, and cancel the tracking of live messages.
  • Message View page, you can trace a tracked message and cancel the tracking of live messages. 
  • Message View page, you can activate Twin View to compare a tracked message leaving Rhapsody with the response from the external system, or vice versa.
