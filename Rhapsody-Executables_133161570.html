Rhapsody Executables

The following executables are located in the <RhapsodyInstallDirectory>\Rhapsody\bin directory:

Windows

File

Description

monitor-start.bat

Starts the Rhapsody Service Monitor on the Windows速 taskbar.

monitor-stop.bat

Stops the Rhapsody Service Monitor on the Windows速 taskbar.

restore.bat

Restores a backup (Administration, Configuration and Message Store) of Rhapsody. Also accessible from the Management Console (Server Settings>Backup). Refer to Backup Areas to learn about the types of information that Rhapsody can back up.

RhapsodyMonitor.exe

The Rhapsody Service Monitor.

RhapsodyServiceCtrl.exe

Starts and stops the Rhapsody service.

versions.bat

Gets the version of the .jar files.

wrapper.exe

Manages the Rhapsody service

*nix

File

Description

restore.sh

Restores a backup (Administration, Configuration and Message Store) of Rhapsody. Also accessible from the Rhapsody Management Console (Server Settings>Backup). Refer to Backup Areas to learn about the types of information that Rhapsody can back up.

rhapsody.sh

Rhapsody startup script. Starts, stops, restarts and gets the status of Rhapsody.

versions.sh

Gets the version of the .jar files.

Configuration Files

The following configuration files are located in the <RhapsodyInstallDirectory>\Rhapsody\bin directory:

Windows

File

Description

config.ini

Contains configuration entries for the components, modules and bundles in Rhapsody.

NTEventLogAppender.dll

Enables Log4j to log events to the Windows速 Event Log.

NTEventLogResource.dll

Contains the Event Log resources for the notifications that go to the Windows速 Event Log.

wrapper.conf, wrapper-local.conf, wrapper-platform.conf, wrapper-license.conf

Java Service Wrapper configuration files. The Java Service Wrapper manages the startup and shutdown processes for the JVM, provides logging, and provides automatic restarts if the Rhapsody JVM fails. Refer to Java Service Wrapper for details.
