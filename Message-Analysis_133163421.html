Message Analysis

The File Manager provides the ability to adapt a message definition using a sample of messages. This simplifies the process of introducing new message types into Rhapsody and enables you to adapt and tune a message definition automatically based on a set of existing messages.

To adapt and tune a message definition:

  1. In the File Manager, select the definition you want to update.

  2. Right-click and select the Adapt to Messages command from the popup menu. The Message Analysis dialog is displayed:

  3. Select the Browse... button next to the Message Folder field to display the Browse for Folder dialog.
  4. Browse to the folder (directory) where your example messages are located and select the OK button to return to the Message Analysis dialog.

  5. Select the Analyze button to start the message analysis. The sample messages are then analyzed and a progress bar is shown as messages are analyzed:

  6. When all messages have been analyzed, a summary of the analysis and changes are shown in the Analysis and Changes panel:

  7. Check the validation errors and select the action to take.
  8. If you want to make a backup of the existing definition before the changes are applied:
    1. Select the Save backup before saving changes to the definition field.
    2. Click the Browse... button next to the Backup Directory field to display the Browse for Folder dialog. Browse to the folder (directory) to which you want to back up the definition and then select the OK button to return to the Message Analysis dialog.
  9. Select the Save button to save your changes to the message definition. The Change Log dialog is displayed, which displays a list of all the changes that will be made to the definition.
  10. If you want to save a copy of the Change Log:
    1. Click the Save Change Log link at the bottom of the screen. The Save As dialog is displayed.
    2. Select the location to which you want to save the change log and then select the Save button.
  11. Select the OK button to save the changes to the message definition. The message definition is updated and the changes are saved.
