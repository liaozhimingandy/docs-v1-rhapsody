Edit Menu

Using the Edit menu, you can search for components and mappings, and copy and paste selected components.

Option

Icon

 Keyboard Shortcut

Description

Copy

Ctrl+C

Copy:

  • Components in your workspace.

  • Filters and connectors on a route.

    Large routes or configurations may take a few minutes to copy depending on the performance of the machine running the engine, and the size of the configuration. A progress bar is displayed to show the time remaining for the operation. 

    If you copy a folder, all components within the folder are copied.

Paste

Ctrl+V

Paste components that have been copied to the clipboard.

Find Component...

Ctrl+F

Locate components in Rhapsody by searching for a keyword.
Search in Mappings...

Ctrl+Shift+F Open the Mapper Search window.
