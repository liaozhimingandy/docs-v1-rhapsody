Managing Routes

Menu Path: Insert>New Route...

Toolbar:

To manage routes, refer to:

  • Creating a New Route
  • Configuring a Route
  • Copying or Moving a Route
  • Deleting a Route

Creating a New Route

To create a new route:

  1. In the Rhapsody IDE Workspace, select a locker or folder you want to create the route in.

  2. Open the Create New Route dialog using one of the following actions:

    • In the right-click menu, select New Route....

    • Select the Route iconin the Rhapsody IDE toolbar.

    • From the Rhapsody IDE Insert menu, select New Route....

    • Press Ctrl+R.

  3. Enter the following details for your new route:

    Field Description
    Template

    Select the template from the drop-down list, or None if you do not want to use a template.

    Refer to Templates for details on how to manage templates.
    Name The name for the new route.
  4. Select the OK button to save your changes, or the Cancel button to discard them.

Configuring a Route

To configure a route's properties, open the filter's Properties dialog by using one of the following methods:

  • Right-click on the route canvas in the Route Editor and select Route Properties.
  • Right-click on the route in the Workspace and select Properties.
  • Right-click on the route in the Object Browser and select the Modify Properties button.

Refer to Configuring Routes for details on the route properties you can configure.

Copying or Moving a Route

To create a copy of a route, including its configuration:

  1. Select the route to copy in the Workspace.
  2. From the right-click menu, select Copy (or press Ctrl+C).
  3. Right-click on a locker or folder you want to copy the route to, and select Paste (or press Ctrl+V).

To move a route to another folder or locker, simply drag-and-drop it to the desired location in the Workspace.

Rhapsody automatically detects the message definitions and tracking schemes that the route uses, and gives you the option to also copy them to the destination locker. Refer to Copying or Moving Components for details.

Deleting a Route

To delete a route:

  1. Select the route you want to delete in the Workspace.
  2. From the right-click menu, select Delete Route from the context menu (or press Del).
  3. Select the Yes button in the confirmation dialog to delete the route, or the Cancel button to abort.
