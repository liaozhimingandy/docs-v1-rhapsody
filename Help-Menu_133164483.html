Help Menu

From the Help menu, you can display Rhapsody's online product documentation, launch the Rhapsody's Support Tracker website, manage your Rhapsody licenses, and so on.

Option

Description

View Help...

Display Rhapsody's online product documentation for your version of Rhapsody in a web browser. You can also view help for a selected item by pressing the F1 key.

If for any reason you cannot access Rhapsody's online product documentation, you will be redirected to navigable offline product documentation (stored in your Rhapsody installation directory).

To change the default help option from online to offline, refer to Help Options.

Rhapsody Support...

Open Rhapsody's Support Tracker website in your browser.

Introduction

Display the Rhapsody Start screen in Rhapsody IDE:

About Rhapsody IDE...

Displays the About Rhapsody dialog, which identifies the version number of your Rhapsody installation, along with the license details.

  • Click licenses@rhapsody.health to send an email to Rhapsody Support requesting a license.
  • Select the Application DLLs button to display a list of installed DLL files and their version. This information may be helpful when contacting Rhapsody Support.
  • Select the Install License button to install a new license. Refer to Installing a New License for details.
