Creating the Web Service Host

  1. Click the New Communication Point icon on the standard toolbar to create a new Web Service Hosting communication point called Patient Lookup Server Web Service Hosting.
     

     
  2. On the General tab, change the Connection Mode field to In->Out.
  3. On the Configuration tab, click the Edit Communication Point Configuration link.

  4. In the Web Service field, select PatientLookupService.

    Refer to Web Service Hosting Communication Point for field definitions.

  5. Click the Add Operation button. Select lookupById (Request-Response), then click the OK button.
     

     
  6. Click the OK button on the Operation Parameter Mapping screen.
     

     
  7. Repeat these steps to add another operation, lookupByName (Request-Response).
     

     
    The Operations tab should now list two operations.
