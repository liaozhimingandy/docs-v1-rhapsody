Managing RPG Private Keyrings

Viewing RPG Private Keyrings

To view the RPG private keyrings registered on the Rhapsody server, navigate to View>Certificate and Key Manager to display the Certificate and Key Manager and select the RPG Private Keyrings tab:

The tab provides the following information:

Field
Description
Alias

The unique user-defined name for the keyring.
Primary User ID The primary user ID for the keyring.
Key Count The key count.

You can perform the following actions from the tab via buttons or the right-click menu:

Action
Description
Import... Add an RPG private keyring.
Export... Export an RPG private keyring.
Remove Delete an RPG private keyring.
Details Display the details for an RPG private keyring.
Show Uses Open the Security Object Uses dialog to display the components an RPG private keyring is being used in. Double-click a component to view or edit its configuration properties.
Filter Perform text-based filtering to filter the list of displayed RPG private keyrings.

Importing an RPG Private Keyring

To import an RPG private keyring:

  1. Navigate to View>Certificate and Key Manager.

  2. On the RPG Private Keyrings tab, select the Import... button. The Import RPG Private Keyring dialog is displayed:

  3. Enter: 
    • A name in the Key Alias field to identify the keyring, if required.
    • The path of the keyring you want to import in the Filename field or click the Browse link to locate it.
    • The passphrase in the Passphrase field.
  4. Select the OK button to import the keyring.

  5. If the import is successful, then the new keyring is added to the list. If it fails, an error message is displayed indicating the reason why it failed.

Exporting an RPG Private Keyring

The 'Export RPG private keys' access right is required to export RPG private keyrings. By default, this privilege is only given to the root Administrator user.

To export an RPG private keyring:

  1. Navigate to View>Certificate and Key Manager.

  2. On the RPG Private Keyrings tab, select a keyring to export and then select the Export... button. The Save As dialog is displayed.

  3. Enter a path in the Filename field to select a path to save the keyring to that location.
  4. Enter a file type in the Save as type field.
  5. Select the OK button to export the keyring. The OK button becomes available only when the mandatory fields have been entered (all fields are mandatory).

Removing an RPG Private Keyring

To remove an RPG private keyring:

  1. Navigate to View>Certificate and Key Manager.

  2. On the RPG Private Keyrings tab, select the keyring you want to remove.

  3. Select the Remove button. A confirmation dialog is displayed.

  4. Select the Yes button on the confirmation dialog to remove the keyring (you can undo the deletion by selecting the Undo Remove button).

Displaying RPG Private Keyring Details

You can examine for a RPG private keyring in the RPG Keyring Details dialog:


The RPG Keyring Details dialog displays the Alias, User ID and Type for the keyring and following information for each of its associated keys:

Field
Description
Key ID The key ID.
Signing Key Whether the key can be used for signing.
Encryption Key Whether the key can be used for encrypting.
Algorithm The encryption algorithm.
Bit Strength The bit strength of the encryption algorithm.
Expires When the key expires. Expired keys are displayed in red.
Created When the key was created.
Users The users associated with the key.

To display the details for an RPG private keyring:

  1. Navigate to View>Certificate and Key Manager.
  2. On the RPG Private Keyrings tab, select the keyring you want to examine.
  3. Select the Details button. The RPG Keyring Details dialog is displayed.

  4. Select the Close button to close the dialog.
