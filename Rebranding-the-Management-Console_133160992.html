Rebranding the Management Console

Custom Enhancements

Partners and resellers can change the image as well as the text styling on the Management Console login page.

Image

If a custom image is not provided, the default Rhapsody login page will be displayed.

  • The maximum image file size is 250 x 122 pixels. Your custom image is displayed 'as is' if it is smaller than the dimensions required, and 'reduced to fit' proportionally if it is larger.
  • The filename of the image must have the name splash-image, and be of type .png, .bmp, .jpg or .gif (for example, splash-image.png).
  • Ensure that you only have a single image file with the name splash-image in the <RhapsodyDataDirectory>\monitoring\splash directory.

Text Styling

You can customize the styling of the text displayed over the image. If no styling information is provided, the default text styling will be used.

The styling must be specified in a CSS file called splash-style.css:

Example: splash-style.css 
#rhapsody_engine_name {
    color: black;
    font-size: 1.5em;
}

CSS Element

Description

rhapsody_engine_name

Name of the engine instance

Customizing the Management Console Login Page

  1. Locate the <RhapsodyDataDirectory>\monitoring\splash directory. This folder is automatically created after the engine starts successfully, when the Management Console is first started. If it does not exist, either start the engine or create the directory manually.
  2. Copy your custom image and the splash-style.css file to this directory.

  3. Start the engine and go to the Management Console login page to view your changes.

    If the CSS or image is updated while the engine is running, refresh the login page to view your changes.
