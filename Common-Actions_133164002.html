Access to Doki

Documentation Help enables you to connect to the Rhapsody Documentation Wiki (Doki). When enabled, the Documentation Help icon  is displayed in the top right-hand corner of Management Console pages:

When you click the icon, a new browser tab opens and connects to the documentation home of the Rhapsody Interface Engine at https://doki.orionhealth.com/display/public/RHAP/Home.

You can configure the behavior of the Documentation Help through the following properties in the rhapsody.properties file:

# Configure the enable state of the documentation icon in the the web management console
#WebMonitorService.documentationHelp.enabled=true
# Configure the URL to link to the documentation help page  
#WebMonitoringService.documentationHelp.url=
# Configure the tooltip text showing on the documentation help icon
#WebMonitoringService.documentationHelp.tooltip=

Changing Documentation URL

You can change the default documentation URL by setting the WebMonitoringService.documentationHelp.url property:

Property Value

Description

Blank value (default).

Default documentation URL.

Entry missing or commented out.

Default documentation URL.

Entry present with a valid URL.

The value overrides the default documentation URL.

Changing the Documentation Tooltip

You can change the default tooltip on the Documentation Link icon  by setting the WebMonitoringService.documentationHelp.tooltip property:

Property Value

Description

Blank value (default).

Default tooltip to connect to the online documentation.

Entry missing or commented out.

Default tooltip to connect to the online documentation.

Entry present with non-blank value.

The value overrides the default tooltip text.

Enabling and Disabling the Help Documentation

To enable or disable Documentation Help, set the WebMonitorService.documentationHelp.enabled property:

Description

Managing Tables

Sorting by Column

To sort a table by column:

  • Click the column's heading name.
  • Re-click the column's heading name to toggle between ascending and descending order, denoted by theandicons, respectively.

If you are monitoring components, ensure you are viewing the table in list view.

Configuring Columns

To configure which columns to display in the search table, click the Add icon  to display the Display Column dialog:

  • Select the columns you want to be displayed in the search table.
  • Deselect the columns you want to be hidden in the search table.

Fields that are greyed out are required fields, and cannot be hidden.
