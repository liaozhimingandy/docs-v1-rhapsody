X12 Project Editor

The X12 Project Editor enables you to view and edit the structure of an X12 definition.

The X12 Project Editor is composed of three tabs:

Tab
Description
Definition

Select a transaction set in order to view or edit its definition.
Composite Dictionary View or edit composite data structures.
Data Element Dictionary View or edit the X12 Data Element Dictionary which defines the data elements.
Rules Enable or disable validation rules.
