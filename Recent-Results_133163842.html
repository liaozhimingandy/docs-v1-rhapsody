Recent Results

Menu Path : Find Messages>Recent Results

Read Access Rights:

  • Log in to Management Console
  • View locker
  • Search message queues
  • Search Error and Hold Queues

The Recent Results page displays a list of the 10 most recent searches.

When you view a recent search again, the same results are shown as per the search criteria you selected, capturing the same column and sort order, and any text entered in the Filtering Bar fields. You can then choose to run the search again as usual. Note that if your permissions have changed since you last ran the search, the search criteria will be updated to display only what you currently have permission to view.

Column

Description

Name

Displays the search name from either the Input, Output, Route, Error Queue or Hold Queue.

If no search name has been entered, the name will default to either Input, Output, Route, Error Queue or Hold Queue, followed by an incrementing numeral if there are multiple recent searches.

Message Count

Number of messages found in the search. Click the View link to display the list of messages found during that search.

Search Completed

Displays how long ago the search was completed.

A Cancel link is displayed for any searches still in progess, enabling you to cancel them. Only the messages found up to the point of cancellation are stored as search results.

To delete searches, select them by checking their checkbox and selecting the Delete button.
