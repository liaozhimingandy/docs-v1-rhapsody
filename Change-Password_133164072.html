Change Password

Menu Path : Management>Change Password

Write Access Rights:

  • Change password
 

The Change Password page enables you to change your password to access the Management Console.

Field

Description

Old Password

Enter your old password for accessing the Management Console.

New Password

Enter your new password to access the Management Console.

Confirm New Password

Re-enter your new password to access the Management Console.

Help Text

This text states the password policy.

Error Handling

There are four possible error types corresponding to the following error messages which appear when you select the Change Password button:

Error Message

Error Reason
