Creating the Web Service Client

Click the New Communication Point icon on the standard toolbar to create a new Web Service Client communication point called Patient Lookup Web Service Client.

Complete the following:

  • General Tab
  • Configuration Tab
    • Web Service Consumer Communication Properties: General Tab
    • Web Service Consumer Communication Properties: Security Tab
    • Web Service Consumer Communication Properties: Operations Tab (Add Operation lookupById)
      • Operation Editor: Request Tab
      • Operation Editor: Response Tab
      • Operation Editor: Fault Tab
    • Web Service Consumer Communication Properties: Operations Tab (Add Operation lookupByName)
      • Operation Editor: Request Tab
      • Operation Editor: Response Tab
      • Operation Editor: Fault Tab

Operation Editor: Fault Tab

Field

Description

Mode

Execute Custom JavaScript When SOAP Fault Is Received

JavaScript

 
// Check the fault code
if (fault.faultCode.localName == 'Receiver') {
	// Treat as a connection error
	fault.treatAsConnectionError();
}

// Insert fault details into message
default xml namespace = "http://www.w3.org/2003/05/soap-envelope";
output.text = "Header|Error\rError|||\r";
output.setField("Error/Code", fault.envelope..Code.Value);
output.setField("Error/SubCode", fault.envelope..Subcode.Value);
output.setField("Error/Message", fault.envelope..Reason.Text);
