Login Settings

Menu Path : Management>Login Settings

Read Access Rights:

  • Log in to Management Console

Write Access Rights:

  • Configure Introduction page

The Management Console includes the option to configure an Introduction Page and a Welcome Message that are displayed to all users prior to or during logging in.

Configuring an Introduction Page

The Introduction Page panel gives you an option to configure the introduction page displayed to a user accessing the Management Console before they see the Login screen. Its purpose is to convey information to the user prior to them logging in, for example, warning them of a change in policy relating to the use of the application or that a network outage has been scheduled.

To configure an information banner:

  1. Enter the following information:

    Field

    Description

    Introduction Page

    Select the On or Off buttons to enable or disable the introduction page. This is set to Off by default.

    Browser Window Title

    Enter the browser title (not mandatory). The entered text here will appear as the name of the browser window.

    Content

    Enter the message you wish to convey. Standard HTML tags are supported.

    Include Link to Login Page

    Select the Yes button to include a link to the login page on the introduction page. If No is selected, then the Content field must include a link to allow the user to login, for example <a href="/rhapsody/index.jsp">Continue to Rhapsody Login</a>.
  2. Select the Apply button to save the changes made to the page.

  3. Click the Preview link to check if the page displays as you intended.

  4. Click the Discard Changes link to discard any changes made to the page.

Configuring a Welcome Message

The Login Page panel enables you to display a welcome message on the Management Console Login window.

To configure a welcome message:

  1. Enter the following information:

    Field

    Description

    Welcome Message

    Select the On or Off buttons to enable or disable a welcome message on the Login page.

    Message

    Enter the welcome message to be displayed on the Login page. Standard HTML tags can be used in the welcome message.

    Any new lines in the welcome message text will be converted to <BR> tags.
  2. View the message in the Preview field to check if the page displays as you intended and click the Apply button to save the changes made to the page.
  3. Select the Apply button to save the changes made to the page, or click the Discard Changes link to discard the changes made.
