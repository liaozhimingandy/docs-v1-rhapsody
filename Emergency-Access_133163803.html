Emergency Access

Rhapsody supports user-initiated 'break-the-glass' functionality, where Rhapsody is able to grant an identified set of authorized users access to electronic health information during an emergency.

An 'authorized' user is one who has the 'View messages in an emergency' access right, but otherwise does not have permissions to view messages or parts of messages. Authorized users, therefore, can access sealed electronic health information but only during an emergency situation. Authorized users are presented with a button for accessing sealed portions of a message, which when selected refreshes the page to reveal the sealed message information. The message is visible to that user for a period of one hour whereafter it is automatically sealed again. 

An 'unauthorized' user, one who does not have the 'View messages in an emergency' access right, cannot access sealed electronic health information at any time.
