Access Control

The access control service protects critical resources in a system by limiting access to only authorized users and their applications. The service prevents the unauthorized use of a resource or the use of a resource in an unauthorized manner.

This is achieved in Rhapsody through the use of user permissions to access features and compartmentalizing portions of the Rhapsody configuration through lockers associated with user group memberships.
