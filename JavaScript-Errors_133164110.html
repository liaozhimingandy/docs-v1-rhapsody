JavaScript Errors

Cannot Find Function

Encountering a processing error of the form "Cannot find function <f> in object <O>" when running JavaScript indicates that the function does not exist in Rhapsody. This may be due to a misspelled function name in the code, or because the function is not supported by the version of JavaScript Rhapsody is using.
