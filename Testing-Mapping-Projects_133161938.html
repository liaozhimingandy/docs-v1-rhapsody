Testing Mapping Projects

Required Access Rights:

  • Login to IDE
  • Make changes with IDE
  • Locker view
  • Run Intelligent Mapper tests

The Mapping Project Testing feature enables you to test the configuration of a mapping project before you implement it in an Intelligent Mapper filter. Using this feature, you can:

  • Create tests to send test a mapping project with messages.
  • View test results.
  • Save tests with the filter configuration, so they can be used again the next time you want to run a test, or change the filter configuration.
