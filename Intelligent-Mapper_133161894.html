Intelligent Mapper

Menu Path : View>Intelligent Mapper Projects...
Toolbar: >Intelligent Mapper Projects

Read Access Rights:

  • Log in to Rhapsody IDE
  • View locker

Write Access Rights:

  • Make changes with Rhapsody IDE
  • Edit locker
  • Run Intelligent Mapper analysis

The Intelligent Mapper is a tool that determines the transformation (mapping) rules between input messages and output messages. It can, for example, assist customers in migrating mappings to Rhapsody from other mapping systems, such as legacy integration engines. The tool implements mapping through mapping projects which are stored on the engine. The mapping projects are used to generate JavaScript mapping code from mapping rules. The mapping code is used by the Intelligent Mapper filter to perform the actual mapping in a Rhapsody route.

The Intelligent Mapper consists of the following components:

  • Mapping Project Manager – enables you to create, edit and delete mapping projects.
  • Mapping Project Editor – enables you to analyze messages, review mapping rules, and generate mapping code.
  • Intelligent Mapper filter – a filter linked to a mapping project that executes mapping code.
