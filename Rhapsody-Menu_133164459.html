Rhapsody Menu

Using the Rhapsody menu, you can check out or check in a component (or the engine), generate documentation for selected components, change or encrypt your password, define how Rhapsody IDE displays details, reload a configuration and so on.

Option

Icon

 Shortcut

Description

Check In All Items

Check in all checked out components or select the components you want to check in.

Check Out


Check out the selected component, and lock it for editing. Other users will be able to view, but not change the configuration until the component is checked in again.

Checking out a selected route or communication point does not put it into a stopped state. This means Rhapsody continues processing messages using the old configuration until any changes are checked in.

Check In

Check in a selected checked out component or select the components you want to check in.

Undo Check Out

Cancel a check-out of a selected component or components, and reverts the configuration back to its original state. This action must be confirmed.

Save Documentation...


Generates a PDF of the configuration of a selected component, or of the engine itself.

User Manager

Ctrl+Shift+U

Launch the User Manager, which is used to maintain the list of Rhapsody users and their access to the application. When a user logs into the Rhapsody IDE or Management Console, their possible actions are limited by the privileges assigned to them by the administrator.

Users without suitable privileges are prevented from configuring the engine, viewing engine statistics, or error messages.

Change Password...



Change your password.

Encrypt Password



Encrypt your password if you do not want your password stored in plain text.

Rename Rhapsody Engine...



Change the name of this instance of the Rhapsody engine.

Server Actions>Stop Rhapsody Engine...



Stop the Rhapsody engine. This action must be confirmed. Stopping the engine closes the Rhapsody IDE and Management Console. It also clears the message count in the current Rhapsody session. Refer to Running the Rhapsody Engine for details on restarting the Rhapsody engine.

Reload Rhapsody Configuration



Reload the current configuration from the engine.

Reloading overwrites any recent changes you have made and have not checked in.

If you want to load an RLC file from another engine, you should use File>Open instead. Refer to Loading a Configuration for details.

Rhapsody Options...



Enable you to:

  • Configure how Rhapsody IDE displays information.
  • Define the default local definition storage location.
  • Enable debugging.
Check-in Comment Rules

Configure check-in comment rules.
