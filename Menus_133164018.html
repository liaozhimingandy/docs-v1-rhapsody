Menus

The Management Console menus are located on the left pane, and are grouped according to their function:

  • Monitoring
  • Find Messages
  • Notifications
  • Management
  • System State
  • Reports

Monitoring

Name

 Keyboard Shortcut

Description

Engine Monitor

 Ctrl+Alt+M

Displays a summary of the Rhapsody engine statistics and lists the watchlists and the status of their associated routes and communication points.

Communication Points

 Ctrl+Alt+P

Lists all communication points, and their current status and activity level.

Routes

 Ctrl+Alt+R

Lists all routes and their current status and activity level.

Monitoring Web Services

Lists all web services and their current status.
Monitoring REST Clients
Lists all REST clients, and their current status and activity level.

System Monitor

 Ctrl+Alt+S

Displays an overview of the system including a graphical history of the engine's message throughput, memory usage over a configurable period of time and disk space usage.

Error Queue

 Ctrl+Alt+E

Displays messages that have been routed to the Error Queue filter or the Error connector of any filter.

Hold Queue

 Ctrl+Alt+H

Displays messages that have been routed to the Hold Queue filter.
Ports in Use
Displays a list of ports being used by the engine and standard components in the engine.

Find Messages

Name

 Keyboard Shortcut

Description

Input Search

 Ctrl+Alt+I

Enables users to search for messages sent from a named input communication point.

Output Search

 Ctrl+Alt+O

Enables users to search for messages that have left Rhapsody.

Route Search

 Ctrl+Alt+T

Enables users to search for intermediate messages within Rhapsody at the point they are received by each route that processed them.

Saved Searches

 Ctrl+Alt+Y

Enables users to view all their saved searches and searches that have been shared with them by other users.

Recent Results

 Ctrl+Alt+U

Displays the most recent searches a user has run.

Notifications

Name

 Keyboard Shortcut

Description

My Notifications

 Ctrl+Alt+N

Enables users to configure a current users notification preferences including delivery addresses, notification level, leave and an overview of components or watchlists a user is subscribed to.

Watchlists

 Ctrl+Alt+W

Enables users to configure watchlists, which are logical groupings of components that provide a visual indicator of any alerts occurring on an individual component. Watchlists also provide a scheduling mechanism that allows redirection of notifications to different users based on their defined shifts.

Suspend / Resume All

Enables an administrator to temporarily disable the sending of notifications for all components and the system.

Delivery Methods

Enables an administrator to set up details for the universal delivery of notifications.

Public Holidays

Enables administrators to specify the days that a public holiday occurs.

Default Settings

Enables users to set up notifications for unmonitored components, set the default Resend and Escalation periods and set the default threshold values for component notifications.

Management

Name

 Keyboard Shortcut

Description

Change Password

Enables users to change their password to access Management Console.

Reset Counters

Enables users to reset the user-specific message counts for component types (communication points, routes or web services).

Monitoring Lookup Tables

Lists the lookup tables available in Rhapsody.

Archive Cleanup

Enables users to schedule archive cleanup, the mechanism by which messages are cleaned up.

Backup and Restore

Enables users to create and manage the backup schedule for the Rhapsody message store.
Product Usage
Enables users to configure whether to send data collection information to Rhapsody.

Message Banner

Enables an administrator to configure a banner to provide useful information to users, or to inform them of any upcoming events of significance.

Login Settings

Enables users to configure an Introduction Page and a Welcome Message that is displayed to all users prior to or during logging in.

General Settings

Enables users to set and display the engine name, search result limits and the session timeout for a user in the Management Console.
Copyright Notices
Displays mandatory copyright notices.

System State

Name

 Keyboard Shortcut

Description

System Log

 Ctrl+Alt+L

Enables users to view and search for log events on a specific component within a specified date and time.

Audit Log

Enables administrators to view all audit events between a specified time period or for a specific user.

Threads

Provides both a view of how Rhapsody operates and recovery control in the event of serious failures, such as runaway routes or hung communication points.

Export Tools

Provides tools for exporting system state information and generating thread dumps, useful when diagnosing a problem.

User Sessions

Lists the users who are currently logged on to the Management Console.

Database Key Values

Displays a table that lists all key values used by the engine.

JavaScript Counters

Enables allocation of unique (integer) identifiers.

Reports

Name

 Keyboard Shortcut

Description

Message Latency

Enables a user to generate and view a report that is a measure of the time a message spends inside the Rhapsody engine.

Response Latency

Enables a user to generate and view a report that displays a measure of the elapsed time before the response of an external system to a message sent to it.

Performance Statistics

Are used to identify bottlenecks during the message processing by the Rhapsody engine.

Message Breakdown

Enables a user to generate and view a report that is a combined chart and table showing the number of processed messages of each type, or containing a specified property.
Engine Uptime
Enables a user to generate and view a report on the up and down times of the engine, as well as the number of sent, processed and received messages for each up time.
