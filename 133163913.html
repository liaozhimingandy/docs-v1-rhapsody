Suspend / Resume All

Menu Path : Notifications>Suspend / Resume All

Write Access Group Rights:

  • Log in to Management Console
  • Suspend/resume component and issue notifications
  • Suspend/resume global notifications

From the Suspend / Resume All page, an administrator can temporarily disable the sending of notification messages for all components and the System. 

Suspend All Notifications

To suspend all notifications:

  1. Enter a time (in minutes) in the Suspend for field and click the Suspend Now button. A confirmation dialog is displayed.
  2. Select the Suspend Now button (or the Cancel link to abort). Suspension of notification messages for all components and the system will commence at the end of the time specified.

  3. In addition to a Resume Notifications button, the following message appears: All notifications have been suspended successfully.

    When notifications are suspended, they can either be resumed or the suspension time can be edited by adding a new value in the Suspend for field and then clicking the Suspend Now button.

Resume Notifications

To resume notifications, select the Resume Notifications button. The following message is displayed: All notifications have been resumed successfully.
