Finding and Replacing Lookup Table Values

Finding Lookup Table Values

To find values in a lookup table:

  1. In the Rhapsody Lookup Tables manager, select the View Values button. The View Lookup Table Values dialog is displayed.

  2. Select the Find icon . The Find and Replace dialog is displayed:

  3. In the Find what field, enter the text you want to search for, and:
    • If you want to only search for text that matches the case of the text you entered in the Find what field, select the Match case checkbox.
    • If you want to replace the text only if it is an exact and complete match of the text you entered in the Find what field, select the Match entire cell contents checkbox.

  4. To find the next occurrence of the text, select the Find Next button.

  5. When you have finished finding and replacing lookup table values, close the Find and Replace dialog.
  6. Select the OK button in the View Lookup Table Values dialog to close it.

You can also use the Edit Lookup Table Values dialog to find look up table values. Refer to Replacing Lookup Table Values for details.

Replacing Lookup Table Values

To find and replace values in a lookup table:

  1. In the Rhapsody Lookup Tables manager, double-click a table to open it for editing. The Edit Lookup Table Values dialog is displayed.

  2. Select the Find / Replace icon . The Find and Replace dialog is displayed:

     

  3. In the Find what field, enter the text you want to search for, and:
    • If you want to only search for text that matches the case of the text you entered in the Find what field, select the Match case checkbox.
    • If you want to replace the text only if it is an exact and complete match of the text you entered in the Find what field, select the Match entire cell contents checkbox.
  4. In the Replace with field, enter the text that you want to replace it with.

  5. Do one of the following:

    Click...

    Find the next occurrence of the text

    Find Next

    Replace an occurrence of the text

    Replace

    Replace all occurrences of the text

  6. When you have finished finding and replacing values, close the Find and Replace dialog.
  7. Select the OK button in Edit Lookup Table Values dialog to save your changes, or the Cancel button to discard them.
  8. Select the Check In icon to check in any changes.
