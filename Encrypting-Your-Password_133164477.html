Encrypting Your Password

A number of Rhapsody communication points, including Email Client, HylaFAX Client, Database and (S)FTP Client, require a username and password to be entered as part of their configuration. The password may be entered as plain text, but this can then be viewed by other users who have access to the engine.

Select the Encrypt Password item from the Rhapsody menu to display the Encrypt Password dialog. This enables you to encrypt a password and then copy the encrypted password to the clipboard to be used within the configuration. The encrypted password can then be used in any field where a password is required.
