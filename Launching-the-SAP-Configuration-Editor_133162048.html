Launching the SAP Configuration Editor

Creating a New Configuration

To create a new configuration:

  1. Create a new SAP RFC communication point or filter in the Rhapsody IDE.
  2. Open the properties of the new communication point or filter, and configure the connection settings if manual connection settings are being used.
  3. Apply the message definition that this SAP RFC component will use in the Definition File field, if applicable. Message definitions are only required if the SAP configuration will need to insert or extract fields from the message.
  4. Click the Create Configuration link in the Configuration File field to launch the Rhapsody SAP Configuration Editor.
  5. Refer to Using the SAP Configuration Editor for details.

Editing an Existing Configuration

To edit an existing configuration:

  1. Open the configuration properties of the communication point or filter.
  2. Click the Edit Configuration link to launch the SAP Configuration Editor. Refer to Using the SAP Configuration Editor for details.
