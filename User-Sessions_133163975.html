User Sessions

Menu Path : System State>User Sessions

Read Access Rights:

  • Log in to Management Console
  • View user session

Write Access Rights:

  • Disconnect user sessions

The User Sessions page lists all the users who are currently logged on to the Management Console.

Column

Description

User

The username of a user currently logged in to the Management Console.

Session Time

The length of time the user has been logged in.

Remote Host

The IP address of the machine on which the user is logged in.

Browser

The version details for the browser that the logged-in user is using. Hovering over the browser text displays a tooltip with the full text, for example:
Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.6; en-US; rv:1.9.2.10) Gecko/20100914 Firefox/3.6.10).

Actions
 

To end a user's current session, click the Disconnect link.

You must have the 'Disconnect user sessions' access right to view the Actions column and enable the Disconnect link.

You can cancel a long-running search executed by a user by terminating their session.
