Viewing All Changes

If you want to view all the changes being applied to a sample message by all the change groups, click the All Changes link in the Navigation Panel.

Configuration Element

Description

Sample Input Message Panel

Displays the input sample message.

Sample Result Panel

Displays the input message with changes. The changes are highlighted.

All Changes Link

Click this link at any time to view all changes to the sample message.
