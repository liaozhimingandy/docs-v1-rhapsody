Maximum Result Rows

The limit for the maximum number of rows the ResultSet object for a statement can contain can be set using the maxResultRows attribute.

<statement name="main" maxResultRows="100">

This attribute can be used to protect against unexpected overflow from a SELECT statement. This optional attribute can have any positive value. If the attribute is omitted there is no limit on the number of rows returned.
