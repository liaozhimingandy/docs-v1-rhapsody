Installation Overview

A Rhapsody installation consists of installing the Rhapsody Engine (server) and Rhapsody Integrated Development Environment or Rhapsody IDE (client):

  • Rhapsody Engine - installs the Rhapsody engine and Management Console.
  • Rhapsody IDE - installs Rhapsody IDE, a WindowsÂ® application for configuring the Rhapsody engine to build interfaces to connect different systems.

The Rhapsody Engine and Rhapsody IDE (client) can be installed on different machines.

Installation Files

Registered users can request Rhapsody installation files through Support Tracker.

Licenses

A valid license must be installed to work with the Rhapsody engine. A license can be requested by emailing licenses@orionhealth.com. Refer to Installing a New License for details on how to install the license.
