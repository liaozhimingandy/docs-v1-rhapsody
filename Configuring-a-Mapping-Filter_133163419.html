Configuring a Mapping Filter

The configuration of the filter itself is simple. As the definition for this filter is contained in the mapper definition file, all that is required is to locate this file and load it into the filter.

  1. Open the Mapper Filter Properties.

  2. In the Mapper Definition File field, select the Browse button to display the Select a message definition dialog, shown in the following screenshot. If the map definition is not shown in the message definition list, you must add the map file to the File Manager. Refer to Working with Message Definitions for details.

  3. Select the definition from the list and select the OK button. The map definition is then added to the Mapper Filter.
