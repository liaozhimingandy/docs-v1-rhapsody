Insert Menu

Using the Insert menu, you can insert new components into your configuration:

Option

Icon

 Shortcut

Description
New Locker...

Ctrl+Y Create a new locker and adds it to the Workspace. Lockers enable you to organize sections of configuration and configure permissions on this section.

New Folder...

Ctrl+T

Create a new folder and adds it to the Workspace. Folders enable you to organize routes and communication points on your workspace.

New Route

Ctrl+R (Ctrl+Shift+R)

Create a new route and adds it to the Workspace. The new route is displayed in the Route Editor where you can configure the route. Refer to Configuring Routes for details.

New Communication Point...

Ctrl+E (Ctrl+Shift+C)

Create a new communication point and adds it to the Workspace. Refer to:

  • Managing Communication Points for details on how to create a communication point.
  • Communication Point Types for details on communication point types and their properties.
