Creating a Server Route

Click the New Route icon on the standard toolbar, and call it Server Route.

Complete the following:

  • Create Base64 Encoding Filters
    • Filter 1
    • Filter 2
  • Create an HL7 Acknowledgement Generation Filter
  • Configure the Server Route
  • Start Components

Create Base64 Encoding Filters
