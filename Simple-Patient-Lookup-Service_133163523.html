Simple Patient Lookup Service

 

Before you create the Web service, you must set up the following:

  • Create Private Keys and Certificates
  • Add a User to the Web Service User Store
  • Create the Folder Structure for the Communication Points and Routes
  • Load the Message Definition File

Create Private Keys and Certificates

View>Certificate and Key Manager>SSL Private Keys>Generate New...
  1. Create two private key and certificate pairs using Rhapsody IDE, and import them into Rhapsody's certificate store. In the Password field, enter r as the password to encrypt the file containing the private key. Refer to Generating a New Certificate and Private Key for step-by-step instructions.

    Name: PatientLookupServiceClient
