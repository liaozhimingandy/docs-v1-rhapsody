System Log

Menu Path : System State>System Log

Read Access Rights:

  • Log in to Management Console
  • View engine logs

Write Access Rights:

  • Export engine logs

The System Log page enables you to view and search for log events on a specific component within a specified date and time. In addition, the logs can be exported as text using any available locale.

Searching System Logs

To search system logs:

  1. Using the following search filters, search through the system logs:

    Search Filter

    Description

    From

    The date and time criteria to start the search from.

    To

    The date and time criteria to end the search at.

    Log Level

    The log level you want displayed. If you select a log level, the log entries for that log level and higher severity log levels are displayed. The available options are:

    • All - displays all logs (default).
    • Fatal - displays fatal logs.
    • Error - displays error and fatal logs.
    • Warn - displays warn, error and fatal logs.
    • Info- displays info, warn, error and fatal logs.
    • Debug - displays debug, info, warn, error and fatal logs.
    • Trace - displays trace, debug, info, warn, error and fatal logs.
    • Off - no logs are displayed.

    Number of results

    The number of log entries you want displayed on a page.

    Component

    To view the log details for a specific component:

    • Enter the first few letters of the component name in the field, and and then select a component from the drop-down list.
    • Select the Open Folder icon and select a component from the Select Component dialog.
  2. Select the Search button to initiate a search.


Exporting System Logs

To export system logs:

  1. Click the Export Logs link to display the Export Logs dialog:

  2. Specify the From and To dates and times, select a Locale from the drop-down list, and select the Export button.
