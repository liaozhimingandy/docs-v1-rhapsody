Creating a Client Route

Click the New Route icon on the standard toolbar, and call it Client Route. Add the HL7v2.3.1.s3d definition to the route.

Complete the following:

  • Create a Timer Communication Point
  • Create a Sink Communication Point
  • Configure the Client Route
    • Filter 1
    • Filter 2

Create a Timer Communication Point

Click the New Communication Point icon on the standard toolbar to create a new Timer communication point called HL7 Message Timer. This will be used on the client route as the HL7 Message feeder. Ensure that you have a sample HL7 message before continuing (download a sample).

Create a Sink Communication Point

Click the New Communication Point icon on the standard toolbar to create a new Sink communication point called HL7 Message Sink.
