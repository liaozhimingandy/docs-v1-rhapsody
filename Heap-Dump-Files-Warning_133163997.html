Heap Dump Files Warning

Rhapsody generates heap dump files when the JVM runs out of memory. When heap dump files are generated, a warning banner is displayed in the Management Console indicating the number of heap dump files and their total size.

Automatic generation of heap dump files can be disabled by removing the heap dump section in the wrapper.conf file located in the bin directory.
