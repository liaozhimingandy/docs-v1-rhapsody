Software Requirements

The software requirements for installing and using this release of Rhapsody are as follows:

  • Rhapsody Engine
  • Rhapsody IDE
  • Management Console
  • Databases
  • Ports
  • Licensing

Existing customers, who plan to upgrade their Rhapsody installation on an operating system that is no longer supported, should contact Rhapsody Support to determine what course of action to take.

Ports

Rhapsody uses the following ports by default. If these ports are not free, then alternative port numbers must be specified when installing the application.

Default Port

Description

3041

Rhapsody IDE port.

4031

Configuration broadcast/browse port to connect to the Rhapsody engine.

8081

Management Console HTTP port.

8444

Management Console HTTPS port.

8449

Web service API HTTPS port.

You must add the ports to the firewall exclusion list otherwise you may not be able to connect from another machine.
