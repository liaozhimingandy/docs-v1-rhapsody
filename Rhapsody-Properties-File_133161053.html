Rhapsody Properties File

The Rhapsody properties file is a text file called rhapsody.properties that contains properties used to configure the Rhapsody engine. The rhapsody.properties file is read by the engine on startup.

For details on the properties you can configure in the Rhapsody properties file, refer to the relevant feature in the product documentation or the comments in the Rhapsody properties file itself The file is located in Rhapsody's installation directory:

<RhapsodyInstallationDir>/rhapsody/rhapsody.properties

To edit the properties, edit the file directly. The engine must be restarted for any changes to the rhapsody.properties file to be activated.

For details on the rhapsody.properties file and upgrading Rhapsody, refer to Upgrade Strategies.
