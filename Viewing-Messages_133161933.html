Viewing Messages

The Message Viewer enables you to examine the matched messages and the structural differences between them. Displayed on individual tabs matched message pairs, unmatched source messages and unmatched target messages. Selecting a message in any of these lists displays the message, or a message diff in the case of matched message pairs.

Messages are displayed using HL7 escape sequences where applicable.

Rhapsody IDE does not display the content of a sample message if you have deleted or relocated the sample message on disk.

The Message Viewer provides the following information:

Tab Description
Matched A list of matched message pairs.
Unmatched Source A list of unmatched source messages.
Unmatched Target A list of unmatched target messages.

Matched

The Matched tab displays a list of matched message pairs. Matched message pairs are source-target message pairs in which at least one message identifier matches in both the source and target messages.

The Matched tab provides the following information:

Panel Description
Source / Target A list of matched source and target message pairs.
Source Message The content of the source message in a selected matched message pair.
Target Message The content of the target message in a selected matched message pair.

The horizontal scrollbars of Source Message and Target Message panels are synchronized by default. To unsynchronize the horizontal scrollbars, deselect the Synchronize scrollbars checkbox.

To copy a source or target message, right-click on the Source Message or Target Message panel, respectively, and select the Copy Message option.

Message diffs are highlighted as follows:

Highlight Color
Description
Orange A modified element in the source and target messages (where element refers to a segment or composite).
Green An added element in target message.
Red An element in the source message that has been deleted in target message.
None An unmodified element in the source or target message.

Unmatched Source

The Unmatched Source tab displays a list of source messages that do not match any target messages.

The Unmatched Source tab provides the following information:

Panel Description
Source A list of unmatched source messages.
Source Message The content of the selected unmatched source message.

Unmatched Target

The Unmatched Target tab displays a list of target messages that do not match any source messages.

The Unmatched Target tab provides the following information:

Panel Description
Target A list of unmatched target messages.
Target Message The content of the selected unmatched target message.
