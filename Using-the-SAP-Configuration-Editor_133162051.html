Using the SAP Configuration Editor

The SAP Configuration Editor allows the user to configure a single SAP connection and a single Remote Function Call:

The SAP Configuration Editor uses the following icons are used to identify the SAP connection and the Remote Function Call.

Image Description
SAP Connection. Allows the configuration of the connection settings to connect to the SAP server.
Remote Function Call. Allows the configuration of the Remote Function Call to use on the SAP server.

RFC Toolbar

The RFC toolbar is displayed when the Remote Function Call is selected in the SAP Configuration Editor and enables the operations described in the following table to be performed:

Image Description
Starts editing the Remote Function Call. Launches the RFC Editor.
Deletes the Remote Function Call.
