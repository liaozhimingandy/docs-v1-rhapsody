Tutorials

The following tutorials provide a starting point in learning how to use Web Services to solve problems in your organization:

  • A Basic Service to Process HL7 Messages - Uses Rhapsody's Web services to receive an HL7 message and send back an acknowledgment.
  • A Simple Patient Lookup Service - Allows lookups by either patient name or patient identifier.

The tutorials assume a basic understanding of Web services. If more information is required, such as the definitions of specific fields or step-by-step instructions to complete a task, there are links within the tutorials that direct you to the appropriate information.
