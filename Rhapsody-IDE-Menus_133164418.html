Rhapsody IDE Menus

The Rhapsody IDE has the following menu structure:

  • File Menu
  • Edit Menu
  • View Menu
  • Insert Menu
  • Rhapsody Menu
  • Tools Menu
  • Help Menu

The menu options displayed depend on the action taken in Rhapsody IDE. For example, Check Out is not available for a component you already have checked out.

Menu options that are not available for selection are displayed in grey.

There are faster ways of accessing the functions in the menus. For example, pop-up menus or toolbar buttons for frequently-used functions.

You can use the ESC key to cancel the display of pop-up menus.
