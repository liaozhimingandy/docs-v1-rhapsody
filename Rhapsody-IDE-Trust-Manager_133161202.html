Rhapsody IDE Trust Manager

The Rhapsody IDE Trust Manager is a stand-alone application that enables you to view and edit the connection details for trusted Rhapsody engines. It can be launched in one of the following ways:

  • From the WindowsÂ® start menu.
  • From a link on the Rhapsody IDE login form (after clicking the More Options button).
  • From a link on some of the SSL certificate errors shown in the Rhapsody IDE.

You can use the Rhapsody IDE Trust Manager to perform the following tasks:

  • Viewing Rhapsody Engines
  • Adding, Editing or Removing a Rhapsody Engine
  • Verify the Rhapsody Engine Certificate
